# Manual Testing Project: SauceDemo.com

## Objective
Performed end-to-end manual functional and UI testing on saucedemo.com,
covering login, product listing, cart, and checkout flows.

## Summary of Work
- Wrote a test plan defining scope and approach
- Designed 12 test cases covering positive and negative scenarios
- Executed all test cases manually in Chrome
- Found and logged 3 defects using GitHub Issues, including
  screenshots, reproduction steps, and severity ratings

## Results
- 12 test cases executed
- 9 Passed, 3 Failed
- 3 bugs logged (see Issues tab)

## Tools Used
Google Sheets (test case design), GitHub Issues (defect tracking),
Chrome DevTools (basic inspection)
